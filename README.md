## Project Overview

The **Global census analysis report** aims to provide insights on the progression or regression in population numbers in various countries according to census held so far since 1970 till date(2022).Through analysis of the world population data set,we can see population growth in some countries/regions an regression in others,giving us a comprehensive view of the performance of various contries population-wise in the last decades.Insights derived from this analysis will help in evaluation of natality and mortality rates,survival rates,overpopulation,so on and inform decision making process
